# 不良情绪分析小程序 🧠

通过问答引导帮助用户探索情绪背后的真实原因。不是诊断，而是一次和自己的对话。

## 📱 项目介绍

基于 uni-app 开发的微信小程序，采用决策树式引导，从表层感受逐步深入到根本原因，帮助用户自我觉察。

### 核心理念

> 有时候，心情不好是有原因的
>
> 通过一系列问题，帮你梳理情绪背后的真实原因

### 核心功能

- **🌱 决策树引导** - 7 道初始筛查题 → 6 个深度分支 → 18 个灵魂拷问
- **🔍 深度探索** - 每个情绪分支包含 4 道深度探查问题
- **💫 灵魂拷问** - 每个分支的终极 3 问，直面内心
- **📊 智能报告** - 基于答案生成个性化摘要和引导问题
- **🎨 海报分享** - 支持生成精美海报保存或分享
- **💾 隐私保护** - 所有数据本地存储，支持导出和删除

## 🎯 问题结构

### 第一层：初始筛查 (7 题)

| 编号 | 问题 | 分支跳转 |
|------|------|----------|
| Q1 | 你是否会说自己好累？ | 疲劳分支 |
| Q2 | 你是否感到焦虑或紧张？ | 焦虑分支 |
| Q3 | 你是否感到孤独或不被理解？ | 孤独分支 |
| Q4 | 你是否对生活感到迷茫或缺乏方向？ | 迷茫分支 |
| Q5 | 你是否感到愤怒或烦躁？ | 愤怒分支 |
| Q6 | 你是否感到悲伤或低落？ | 悲伤分支 |
| Q7 | 综合检查：以上都不是，但就是心情不好？ | 说不清分支 |

### 第二层：深度分支 (6 个分支)

每个分支包含 4 道深度探查问题，例如：

**疲劳分支**:
- Q1-1: 身体上的劳累 (睡眠不足、体力透支)
- Q1-2: 精神上的劳累 (用脑过度、信息过载)
- Q1-3: 情绪上的劳累 (社交消耗、共情疲劳)
- Q1-4: 意义上的劳累 (觉得做事没意义、空虚)

**焦虑分支**:
- Q2-1: 未来不确定性 (担心未发生的事)
- Q2-2: 过去遗憾或错误 (反刍思维)
- Q2-3: 当下压力 (工作/学业/人际)
- Q2-4: 身体症状 (心悸、失眠、胃痛)

### 第三层：灵魂拷问 (每个分支 3 题)

每个分支都有独特的终极三问，例如：

**疲劳分支**:
1. 你一直在忙，是真的必须忙，还是不敢停下来？
2. 如果停下来，你害怕面对什么？(输入)
3. 这种累，是身体在替你的内心喊停吗？

**焦虑分支**:
1. 你担心的那件事，有多少真的发生了？
2. 焦虑让你感觉在掌控生活，还是在被掌控？
3. 如果最坏的结果发生了，你真的无法承受吗？

### 第四层：补充维度 (4 题)

- E1: 持续时间
- E2: 影响程度
- E3: 应对方式
- E4: 帮助需求 (多选)

## 📊 报告生成

系统内置丰富的文案库，根据用户答案随机组合生成报告：

- **摘要库**: 每个情绪分支 10 条深度摘要
- **问题库**: 每个情绪分支 10 个引导问题
- **海报标题**: 8 种标题样式
- **底部文案**: 8 条温暖寄语

## 🚀 快速开始

### 环境要求

- Node.js >= 16
- HBuilderX（推荐）或 CLI
- 微信小程序开发者工具

### 安装依赖

```bash
npm install
```

### 运行项目

```bash
# 微信小程序 - 开发模式
npm run dev:mp-weixin

# 微信小程序 - 生产构建
npm run build:mp-weixin
```

### 配置小程序 AppID

编辑 `manifest.config.js`，替换为你的小程序 AppID：

```javascript
mpWeixin: {
  appid: 'wxYOUR_APP_ID', // 替换为实际 AppID
  // ...
}
```

## 📖 使用说明

### 1. 开始探索

1. 打开小程序，点击"开始探索"
2. 回答 7 道初始筛查题
3. 根据答案进入对应的情绪分支

### 2. 深度探索

- 回答分支中的 4 道深度问题
- 完成 3 道灵魂拷问（可选输入）
- 填写补充维度问题

### 3. 查看报告

- **答案摘要**: 根据你的答案生成的深度摘要
- **引导问题**: 一个值得思考的问题
- **情绪标签**: 记录探索过程中识别的情绪类型

### 4. 保存与分享

- 点击"保存报告"保存到本地
- 点击"生成海报"生成分享图片
- 点击"重新开始"开始新的探索
