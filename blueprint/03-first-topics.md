# 03. First Topics

## Topic Selection Principle

The first topics should come from real life situations that feel ordinary but carry deeper existential tension.

They should be:

- common
- emotionally recognizable
- not too clinical
- deep enough to create real self-recognition

## Topic 1

### 生活变好后，我为什么还是不快乐

This topic fits users who:

- completed a goal
- improved living conditions
- finally obtained something they expected
- then discovered the inner change was much smaller than imagined

Core mechanism:

- hedonic adaptation
- symbolic expectation overload
- confusing convenience with meaning

Key question direction:

- 你期待的到底是“拥有”，还是“拥有之后的某种自我感觉”？
- 你失望的是现实本身，还是想象没有兑现？
- 你现在缺的不是条件，还是缺一种会被生活触动的状态？

## Topic 2

### 我是在休息，还是在逃避

This topic fits users who:

- stay home often
- feel tired but not restored
- postpone action
- use comfort as a shelter but remain unsatisfied

Core mechanism:

- depletion vs avoidance
- low-energy self-protection
- comfort without renewal

Key question direction:

- 你停下来之后，真的更轻松了吗？
- 你不出门，是因为需要恢复，还是因为不想碰到某种感受？
- 你现在需要的是休息，还是重新接触世界的一点力量？

## Topic 3

### 为什么更容易活，不等于更想活

This topic fits users who:

- improved their environment
- reduced friction in life
- expected better conditions to naturally produce better feeling
- then found daily life only became smoother, not richer

Core mechanism:

- convenience does not equal vitality
- reduced friction can also reduce intensity
- life needs chosen experiences, not only optimized efficiency

Key question direction:

- 方便帮你省下了什么，又让你少经历了什么？
- 你真正怀念的是麻烦本身，还是麻烦里的参与感？
- 你是不是把“更容易活”误以为“会更想活”？

## Topic Priority

Suggested build order:

1. `生活变好后，我为什么还是不快乐`
2. `我是在休息，还是在逃避`
3. `为什么更容易活，不等于更想活`
