# Emotion Analysis Blueprint

This folder stores product blueprints, long-term direction, and future idea notes for the `emotion-analysis` project.

## Files

- `01-system-roles.md`: role split between `mood-diary` and `emotion-analysis`
- `02-trigger-and-routing.md`: how users move from daily logging into deep guidance
- `03-first-topics.md`: the first three deep-analysis topics to build
- `04-tone-and-writing.md`: copywriting and guidance principles
- `05-mvp-scope.md`: MVP page and field changes
- `06-diary-signals.md`: daily signal fields and trigger rules
- `07-bridge-loop.md`: the loop between `mood-diary` and deep guidance
- `IDEAS.md`: future ideas, open directions, and research notes

## Product Direction

The long-term system is:

- `mood-diary` as the lightweight daily entry point
- `emotion-analysis` as the focused deep-guidance layer
- philosophical reflection as a gentle framing, not heavy preaching

The goal is not only to record emotion, but to help users notice numbness, automatic living, and the loss of felt meaning, then re-enter life with more awareness.
